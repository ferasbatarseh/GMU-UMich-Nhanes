The database size is 196 MB.
To download it, follow this link:

https://drive.google.com/file/d/1NBu1CPRgXkprhU7SfuEzwUMrcCD7399r/view?usp=sharing

The download will provide a .BAK file, the file can be restored into SQL server management studio as follows:

1- create an empty database
2- right click and choose restore database
3- select the .BAK file
4- Submit and wait for the database to be built.

For SQL server management studio, use this link:
https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15
